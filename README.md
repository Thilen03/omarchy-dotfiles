# Osaka Jade

Osaka Jade is a theme for Hyprland, Waybar, and Ghostty terminal.

![Preview](preview.png)

## Features

- Waybar
  - Color scheme for Osaka Jade
  - Spotify integration
  - System monitor (CPU, RAM, network)
  - Weather widget

- Hyprland
  - Customized layouts
  - Transparent panels

- Ghostty Terminal
  - Terminal color scheme matching Osaka Jade
